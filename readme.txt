This plugin adds Visage as an oEmbed provider.

== Description ==
This plugin adds Visage as an oEmbed provider so you can share charts in your blog by simply pasting the view URL into the content area of your blog posts.

== Installation ==
Simply install the plugin to your WordPress site, everything else is automatic.
